## 技术说明

### Over Rest 逻辑
```
工作时间内：
  有活跃番茄钟 → FOCUS 状态
  无番茄钟 + 休息 < 5分钟 → REST 状态
  无番茄钟 + 休息 > 5分钟 → Over Rest
    → 宽限期 (默认5分钟)
    → 超过宽限期 → 执行动作
```

### 状态机
```
LOCKED → PLANNING → FOCUS → REST → FOCUS/PLANNING
```
