# Notification Icons

This directory contains icons used for browser notifications.

## Required Files

- `pomodoro.png` - Main notification icon (192x192 recommended)
- `pomodoro-badge.png` - Badge icon for notifications (72x72 recommended)

## Icon Guidelines

- Use PNG format with transparency
- Main icon: 192x192 pixels
- Badge icon: 72x72 pixels
- Use the pomodoro (tomato) theme
