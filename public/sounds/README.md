# Notification Sounds

This directory contains audio files for pomodoro completion notifications.

## Required Files

- `bell.mp3` - Classic bell sound (default)
- `chime.mp3` - Soft chime sound
- `gentle.mp3` - Gentle notification sound

## Recommended Sources

You can obtain royalty-free notification sounds from:
- [Freesound.org](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)
- [Zapsplat](https://www.zapsplat.com/)

## Audio Specifications

- Format: MP3
- Duration: 1-3 seconds
- Sample Rate: 44.1kHz
- Bit Rate: 128-192 kbps
